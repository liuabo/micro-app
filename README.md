# micro-app

## Project setup
```
yarn install
```

### 下载所有应用依赖

```
yarn examples:install

```

### 启动所有应用
```
yarn examples:start
```

### 子应用独立下载依赖 启用
```
yarn install:micro-app-one 
```
```
yarn start:micro-app-one
```
